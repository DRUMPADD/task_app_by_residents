# Aplicación de práctica
## Task app - creada por residentes

## Instalar dependencias 💻
~~~bash
    pip install -r requirements.txt
~~~

## Crear projecto
~~~bash
    django-admin startproject project_name
~~~
## Crear aplicación
~~~bash
    django-admin startapp app_name
~~~

## Ejecutar aplicación 🚀
~~~bash
    python manage.py runserver
~~~
